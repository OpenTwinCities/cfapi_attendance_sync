CfAPI_Attendance_Sync
=====================

A real hacked-together attempt to push Attendence data into the CfAPI. Our 
first goal is Meetup.

## Requirements

- Python
- Virtualenv

## Install

    git clone
    cd cfapi_attendance_sync
    virtualenv env
    source env/bin/activate
    pip install -r requirements.txt
